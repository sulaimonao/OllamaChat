# Future Plans

The project has several exciting features planned for the future. These include:

*   **Improved Tool Use:** We plan to add more tools and improve the model's ability to use them. This includes a calculator, a calendar, and a tool for interacting with APIs.
*   **Multi-modal Support:** We want to add support for multi-modal models that can understand and generate images, audio, and video.
*   **Better UI/UX:** We are constantly working to improve the user interface and user experience of the application.
*   **Advanced RAG:** We plan to implement more advanced Retrieval-Augmented Generation (RAG) techniques to improve the quality of the model's responses.
